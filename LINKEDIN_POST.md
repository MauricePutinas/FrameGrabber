## 🇬🇧 English version

🎬 Sometimes a project just starts for fun.

On the side I'm building a small, playful **desktop‑companion app** – an animated buddy that lives on my screen. For that I needed **clean, transparent videos**. And that's where it gets annoying: the tools for it want a subscription, upload to the cloud, or slap on a watermark.

So I built my own: **FrameGrabber** ⚡

What it does:
🟢 Extract every single frame of a clip, losslessly
🟣 Cut out **real alpha** via chroma‑key (green screen)
🔵 Build finished animations from it: **GIF, APNG & WebP** – with true soft transparency
🧩 Contact sheets & sprite sheets for game/animation workflows
🧠 Auto‑detects the grid **and** the original playback speed

Best part: **if you already own the `.mp4`, it costs €0.** Everything runs locally on your machine – no subscription, no upload, no cloud.

Built with Python, ffmpeg and Pillow. Open source (MIT) on my GitHub.

What started as "I need this for my fun project" became a tool I now use all the time. That's exactly why I love building things myself. 🚀
